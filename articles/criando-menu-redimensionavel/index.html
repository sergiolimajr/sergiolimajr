Criando um menu redimensionável

Estive lendo alguns posts bem antigos do blog Web Design Depot e encontrei o post How to create a resizing menu bar, que é basicamente como criar uma barra de menu redimesionável. A partir disso resolvi refazer o código e trazer para você.

Caso queira dar uma olhada na demo ou repositório no GitHub.

Criando a estrutura do HTML


<!DOCTYPE html>
<html lang="pt-BR">
	<head>
		<meta charset="UTF-8">
		<title> Como criar uma barra de menu redimensionável </title>
		<link rel="stylesheet" href="assets/css/style.css">
    	<link href="http://fonts.googleapis.com/css?family=Merriweather:900,900italic,300,300italic" rel="stylesheet">
		<script src="http://code.jquery.com/jquery-1.11.2.min.js"></script>
	</head>
	<body>
		<header class="grande">
			<nav>
				<ul>
					<li><a href="javascript:;">Home</a></li>
					<li><a href="javascript:;">Services</a></li>
					<li><a href="javascript:;">About</a></li>
				</ul>
			</nav>
		</header>

		<section class="screenplay">
			<p>Scroll nimim até o fim</p>
			<p class="bottom">Fim da linha, camarada.</p>
		</section>

	<script src="assets/js/script.js"></script>
	</body>
</html>

A fonte fica a críterio.

Logo após ter feito essa estrutura, vamos ao próximo passo.

Criar a folha de estilo


* {
    margin: 0;
    padding: 0;
}

header, a, ul, li {
    -webkit-transition: linear all .4s; /* Safari and Chrome */
       -moz-transition: linear all .4s; /* Firefox 4 */
         -o-transition: linear all .4s; /* Opera */
            transition: linear all .4s;
}

body {
    background: #48BEAB;
}

Bom, nessa parte do CSS não tem muito o que explicar. Fiz um reset básico, apliquei uma transição linear em algumas tags e defini uma cor para o fundo da página.



header {
    background: #EBEBEB;
    box-shadow: .0em .0em .3em #3e3e3e;
    opacity: .70;
    width: 100%;
    position: fixed;
    z-index: 10;
}

.grande {
    height: 120px;
}

    .grande ul {
        list-style: none;
        text-align: center;
        margin-top: 45px;
    }

        .grande li {
            display: inline;
            margin: 0 30px 0 30px;
        }

            .grande a {
                text-decoration: none;
                font-size: 1.3em;
                font-weight: 600;
                font-family: 'Merriweather', sans-serif;
                color: #43ACAB;
            }

            .grande a:hover {
                color: #222;
            }

Aqui defini altura de 120px pra o header, posicionei os elementos que estão contidos em nav dentro do header. Lembrando, é necessário usar a classe definida para header ao invés do próprio header, para que o jQuery remova-a e adicione a próxima classe que estamos prestes a criar.


    .pequeno {
        height: 50px;
    }

        .pequeno ul {
            list-style: none;
            text-align: center;
            margin-top: 13px;
        }

            .pequeno li {
                display: inline;
                margin: 0 30px 0 30px;
            }

                .pequeno a {
                    text-decoration: none;
                    font-size: 1.3em;
                    font-weight: 600;
                    font-family: 'Merriweather', sans-serif;
                    color: #43ACAB;
                }

                .pequeno a:hover {
                    color: #222;
                }

section.screenplay {
    float: left;
    height: 500px;
    width: 100%;
}

    section.screenplay p {
        font-family: 'Merriweather', sans-serif;
        font-size: 25px;
        color: #333;
        text-align: center;
        position: relative;
        margin-top: 250px;
    }

    section.screenplay p.bottom {
        top: 100%;
    }

Após ter definido o estilo padrão, defini o estilo a ser mudado pelo jQuery quando for usado o scroll. Criei outra classe, apenas dentro do CSS, chamada pequeno com todo o estilo que será usado ao usar o scroll. É basicamente as mesmas propriedades da classe padrão de header, chamada grande.

Além de criar a nova classe dentro do CSS, reduzi os valores de algumas propriedades como: altura de header para 50px, diminui a margin-top do menu que antes era de 45px, agora para 13px.

Entendendo nosso jQuey


$(document).on('scroll', function() {
    if($(document).scrollTop() > 100) {
        $('header').removeClass('grande').addClass('pequeno');
    }
    else {
        $('header').removeClass('pequeno').addClass('grande');
    }
});

Na função acima usando o evento scroll faremos com que jQuery perceba se o scroll foi deslocado 100px no documento, com isso se o scroll for maior que 100px ele removerá a classe grande e adicionará a classe pequeno. Dessa forma o menu redimensionará mudando o tamanho com a transição linear definida para não ficar com uma mudança rude, se é que me entende.

Conclusão

Você pode inovar, pegando a ideia do código jQuery e implementar em outro tipo de menu ou até criar um novo. É muito fácil. Espero que tenham gostado, obrigado!
